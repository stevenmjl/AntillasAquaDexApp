# 🌊 Antillas AquaDex: Explorando el Corazón del Caribe 🐚

**Antillas AquaDex** es una experiencia digital vibrante y educativa diseñada para dar a conocer la asombrosa biodiversidad marina de la región de las Antillas. Desde las profundidades del arrecife hasta los manglares costeros, esta app combina un catálogo enciclopédico con un sistema de gamificación para fomentar la conservación y el aprendizaje.

---

## ✨ Características Principales

### 📚 Catálogo Maestro de Especies
Explora un archivo detallado de más de **180 especies** clasificadas en 10 categorías taxonómicas:
*   🐠 **Peces** (Óseos y cartilaginosos)
*   🐋 **Mamíferos Marinos** (Delfines, ballenas y manatíes)
*   🐢 **Reptiles** (Tortugas e iguanas costeras)
*   🌊 **Cnidarios y Poríferos** (Corales, medusas y esponjas)
*   🦀 **Crustáceos y Moluscos**
*   ⭐ **Equinodermos y Aves Marinas**

### 🎮 Zona de Trivia Interactiva
Pon a prueba tus conocimientos con un sistema de desafíos diseñado para el aprendizaje:
*   **100 preguntas originales** vinculadas directamente a la base de datos de especies.
*   **Lógica de Re-intento:** Si fallas, pierdes una vida pero permaneces en la pregunta para razonar la respuesta correcta.
*   **Dificultades Dinámicas:** Modo Fácil (20s) y Difícil (40s).
*   **Medallas Coleccionables:** Completa las 10 preguntas de una categoría sin perder tus 3 vidas para ganar insignias exclusivas.

### 🎨 Interfaz de Usuario "Aqua-Inmersiva"
*   **Diseño Temático:** UI inspirada en el océano con degradados marinos, efectos de burbujas y tonos `surfaceColorAtElevation` que simulan la profundidad del agua.
*   **Feedback Visual:** Animaciones de respuesta y transiciones suaves.
*   **Barra de Herramientas Dinámica:** `AquaDexTopBar` con decoraciones de ondas y burbujas.

## 🛠️ Stack Tecnológico

La aplicación está construida con las tecnologías más modernas de Android:

*   **Lenguaje:** [Kotlin](https://kotlinlang.org/)
*   **UI:** [Jetpack Compose](https://developer.android.com/jetpack/compose) (Arquitectura declarativa)
*   **Inyección de Dependencias:** [Hilt](https://developer.android.com/training/dependency-injection/hilt-android)
*   **Base de Datos:** [Room](https://developer.android.com/training/data-storage/room) (Persistencia de trivia, usuarios y favoritos)
*   **Gestión de Imágenes:** [Coil](https://coil-kt.github.io/coil/) (Carga eficiente de imágenes de especies)
*   **Arquitectura:** MVVM (Model-View-ViewModel) + Clean Architecture.
